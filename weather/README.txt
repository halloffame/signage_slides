This Weather page currently uses wunderground's api. See http://www.wunderground.com/.

For this to work, you need to pass the api url into the url as the parameter "weatherapi"

It will look something like this:

  http://halloffame.github.com/signage_slides/weather/?weatherapi=http://api.wunderground.com/api/your_key/forecast/q/CA/La_Mirada.json
